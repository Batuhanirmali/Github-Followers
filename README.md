# Github-Followers
In this training phase, we took the data from github and made an application called followers.


  - Diffable Data Source,
  - Persistence,
  - Costum Alerts,
  - UITableView,
  - Swipe To Delete,
  - Dynamic Type,
  - SFSymbols,
  - UIStackView,
  - DateFormatter,
  - Delegation,
  - SafariVC,
